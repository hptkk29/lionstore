Facebook Pixel

Login to your Facebook account and then go to Events Manager after that click the green "Add" button

- After select Web and click continue

- Select Facebook Pixel and click "Connect"

- Set a name for your Facebook Pixel and click "Continue"

- Now you can see your Facebook Pixel Id, copy it and go to your LION STORE dashboard
https://your-domain.com/dashboard/settings/analytics and paste it in Facebook Pixel Id field
