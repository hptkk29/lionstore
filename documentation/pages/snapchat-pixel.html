Snapchat Pixel

Log in to Ads Manager

- Now click "Create Ads" button and select "Events Manager"

- Click New Event Source and select Create Snap Pixel then Confirm

- Now select Pixel Code and click "Continue"

- Then you can see your Snapchat Pixel Id, copy it and go to your LION STORE dashboard
https://your-domain.com/dashboard/settings/analytics and paste it in Snapchat Pixel Id field
