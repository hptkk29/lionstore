Create a database

After we prepared our server, we can now create a database and import SQL file included

Login to your mysql (root is default username): 
sudo mysql -u root -p
Create a database (change YOUR_DATABASE_NAME with your database name): 
create database YOUR_DATABASE_NAME;
Now run the following command one by one, don't forget to change YOUR_USERNAME_HERE with your database username (any name) and YOUR_PASSWORD_HERE with a strong database password. 
CREATE USER 'YOUR_USERNAME_HERE'@'localhost' IDENTIFIED BY 'YOUR_PASSWORD_HERE';
GRANT ALL PRIVILEGES ON *.* TO 'YOUR_USERNAME_HERE'@'localhost' WITH GRANT OPTION;
FLUSH PRIVILEGES;
Change YOUR_DATABASE_NAME with your database name 
use YOUR_DATABASE_NAME;
Upload "database.sql" file and run the following command 
source /usr/share/nginx/html/database.sql;
Put this code in mysql config file, follow the video above to understand how: (dont forget [mysqld] before) 
[mysqld]
sql_mode = STRICT_TRANS_TABLES,NO_ZERO_IN_DATE,NO_ZERO_DATE,ERROR_FOR_DIVISION_BY_ZERO,NO_ENGINE_SUBSTITUTION
Restart mysql service: 
sudo systemctl restart mysqld
