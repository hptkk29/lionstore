Installation

Now it's time to install LION STORE and make it live, follow this video.
You can find commands used here below

Install node modules: 
npm install
Generate key for your application: 
node ace generate:key
Build LION STORE application: 
npm run build:frontend
npm run build:backend
To extract build folder's contents: 
unzip build.zip
Finally, let's make our application live: 
pm2 start ecosystem.config.js
pm2 save
