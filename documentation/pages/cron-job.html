Cron job

Cron is a service that runs in the background and launches configured tasks on a schedule. This makes it very useful for server maintenance. We need this to automatic check for expired flash deals and coupons

Cron is present on CentOS 8 by default. If for some reason it is not there, then you can install it with the command: 
dnf install crontabs
To start the cron, run the following command: 
systemctl start crond.service
Now let's run it and add it to startup: 
systemctl enable --now crond.service
Then let's run a new cron job: 
crontab -e
Now let's run it and add it to startup: 
0 0 * * * cd /usr/share/nginx/html && node ace scheduler:run >> /dev/null 2>&1
