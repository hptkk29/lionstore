TikTok Pixel

Log in to Ads Manager

- Go to Assets then click Event

- Choose Website Pixel and click "Manage"

- Click Create Pixel button

- Now set a name for your pixel and choose Manually install pixel code and click "Next" button

- Select Standard Mode and click "Next"

- Then you can see your TikTok Pixel Id, copy it and go to your LION STORE dashboard
https://your-domain.com/dashboard/settings/analytics and paste it in TikTok Pixel Id field
