Nginx Configuration

Now it's time to config Nginx, follow this video and you can find codes and commands used below it.

If you already have an SSL certificate use the following code:
lock Nginx config file for SSL

If you don't have an SSL certificate use the following code:
lock_open Nginx config file for no SSL

- You can get a free SSL certificate here: https://zerossl.com

Since we configuring Nginx to the TLS certificates, we are opening port 443/tcp. 
firewall-cmd --add-port=443/tcp --permanent
Now reload firewall 
firewall-cmd --reload
Then restart your Nginx using this command: 
sudo systemctl restart nginx
