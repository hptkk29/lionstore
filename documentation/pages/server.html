Preparing Server

Now let's get our server ready, We are gonna install few packages on the server like MySql, Nginx, NodeJS...
All commands used in this video you can find them below

report Please watch this video before your run any command below

1. Installing Nginx

Run the following commands one by one: 
sudo dnf install nginx
sudo systemctl enable nginx
sudo systemctl start nginx
sudo systemctl start firewalld
sudo firewall-cmd --permanent --add-service=http
sudo firewall-cmd --reload

2. Installing NodeJS

Run the following commands one by one: 
sudo dnf install -y gcc-c++ make
curl -sL https://rpm.nodesource.com/setup_14.x | sudo -E bash -
sudo dnf install nodejs
node -v
npm -v

3. Installing Zip & Unzip

Run the following command: 
yum install zip unzip

4. Installing Redis

Run the following commands one by one: 
sudo dnf install redis nano
sudo systemctl start redis.service
sudo systemctl enable redis
sudo systemctl status redis
sudo firewall-cmd --permanent --new-zone=Redis
sudo firewall-cmd --permanent --zone=Redis --add-port=6379/tcp
sudo firewall-cmd --reload
sudo systemctl restart redis

5. Installing MySQL

Run the following commands one by one: 
yum -y install @mysql
sudo systemctl start mysqld
sudo systemctl enable --now mysqld
sudo systemctl status mysqld
mysql_secure_installation

6. Installing PM2

Run the following command: 
npm install pm2 -g
